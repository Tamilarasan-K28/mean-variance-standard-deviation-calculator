# Mean Variance Standard Deviation Calculator

This project is part of the freeCodeCamp Data Analysis with Python certification.

## Description
The program calculates statistical metrics for a 3x3 matrix.

Metrics:
- Mean
- Variance
- Standard Deviation
- Maximum
- Minimum
- Sum

## Technologies
Python
NumPy

## Example

Input:
calculate([0,1,2,3,4,5,6,7,8])

Output:
Dictionary containing row, column and flattened calculations.
